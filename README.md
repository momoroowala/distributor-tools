# Free Distribution Tools

AI-powered tools for distributors and logistics companies. No signup, no API keys, completely free.

## Tools

### Assessments (PDF Reports)
- **Profit Leak Diagnostic** - Calculate how much profit is silently leaking from your distribution business
- **CEO Time Audit** - See where your hours actually go, and what could be automated
- **Business Dependency Score** - How dependent is your business on you? Find single points of failure
- **AI Readiness Scorecard** - 12-question assessment across Data, Processes, People, and Technology

### Utility Tools
- **RFQ Parser** - Paste a messy RFQ email, get structured data (demo, requires backend)
- **Reorder Point Calculator** - Calculate optimal inventory reorder points

## About

Built by [Automatons.ai](https://automatons.ai) to help distribution businesses identify inefficiencies and opportunities for AI-powered automation.
